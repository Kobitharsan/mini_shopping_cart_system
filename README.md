# Mini Shopping Cart System (CRUD Application)

## Project Description

The **Mini Shopping Cart System** is a responsive web application built using **HTML**, **Tailwind CSS**, and **JavaScript**. It allows users to browse a list of products, add products to a shopping cart, update item quantities, remove items, and automatically calculate the total price. The project focuses on implementing complete CRUD (Create, Read, Update, Delete) operations while strengthening JavaScript fundamentals such as DOM manipulation, event handling, functions, arrays, objects, and application logic.

The user interface should be clean, modern, and fully responsive, with Tailwind CSS used for styling. The application logic should be organized using reusable functions to ensure code readability and maintainability. The project files should be separated into individual files (`index.html`, `style.css` if needed, and `script.js`) following good development practices.
